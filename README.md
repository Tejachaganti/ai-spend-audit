# AI Spend Audit

AI Spend Audit is a web application that helps startups identify overspending on AI tools like ChatGPT, Claude, Cursor, GitHub Copilot, and Gemini.

The platform analyzes current plans, monthly spending, and team usage to recommend cheaper alternatives, optimized pricing plans, and estimated monthly/yearly savings.

---

# Features

- AI tool spend analysis
- Monthly and yearly savings calculations
- Personalized audit summaries
- Shareable audit reports
- Email lead capture
- AI-generated optimization insights
- Public shareable result URLs

---

# Tech Stack

- Next.js 16
- TypeScript
- Tailwind CSS
- Supabase
- OpenAI API
- Vitest
